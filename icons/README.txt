PWA Icons Needed
================
This directory should contain the following PNG icons:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

How to generate:
1. Use a favicon generator like https://favicon.io/
2. Or download free icons from https://www.flaticon.com/
3. Or use ImageMagick/SVG to PNG converter

For now, the Manifest points here but you need real icons.
